CSS POSITIONS

  1. Static:Position based solely on code. Cannot take top/bottom etc. positions
  2. Relative:Position is relative to what the static position would have been
  3. Fixed:Position relative to viewable window
  4. Absolute:Position relative to parent. If no parent, defaults to document (simialr to a static version of fixed.
  5. Sticky:Mix of fixed and relative, depending on the scrolling position.
